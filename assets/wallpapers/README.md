# Wallpapers

Drop your wallpaper files here to use repo-based defaults in `utils/settings.sh`.

Default filenames:

- `desktop.jpg` (desktop + lock screen wallpaper)
- `login.jpg` (GDM login screen wallpaper)

You can also point to custom paths using:

- `LINUX_SETUP_DESKTOP_WALLPAPER_PATH`
- `LINUX_SETUP_LOGIN_WALLPAPER_PATH`
